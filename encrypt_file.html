Encrypt and decrypt a file

Usage

encrypt_file(src, dest, config)


decrypt_file(src, dest, config)

Arguments

src
The source filename (clear file for encrypt_file, encrypted file for decrypt_file).
dest
The destination filename (encrypted file for encrypt_file, decrypted file for decrypt_file). Can be the same as src in which case the file will be overwritten without warning.
config
A encryptr_config object describing the encryption approach to use.

Description

Encrypt and decrypt a file.
