data_request_access(path_data = ".", path_user = NULL, quiet = FALSE)


config_data(path_data = ".", path_user = NULL, test = TRUE, quiet = FALSE)

Arguments

path_data
Path to the data. The default is the current working directory.
path_user
Path to the directory with your user key. Usually this can be ommited. Use the encryptr.user.path global option (i.e., via options()) to set this more conveniently.
quiet
Suppress printing of informative messages.
test
Test that the encryption is working? (Recommended)
