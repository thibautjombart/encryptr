Encryption configuration

Usage

config_sodium_symmetric(path_key)


config_sodium_public(path_pub, path_key)


config_sodium_authenticated(path_pub, path_key)

Arguments

path_key
Path to, or contents of, the symmetric key (config_sodium_symmetric) or your personal private key (config_sodium_public, config_sodium_authenticated). For config_sodium_public, this may be NULL in which case only encryption can be carried out.
path_pub
Path to, or contents of, the public key; your personal public key (config_sodium_public) or the other party's key (config_sodium_authenticated).

Description

Load sodium keys for use with encrypt/decrypt.

Details

Note that the interface here is quite different to that for config_openssl which loads ssh keys; here we expect keys to be provided as raw data, or as links to binary files that contain that raw data. In contrast for openssl we use standard key formats.
